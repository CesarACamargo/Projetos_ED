# 📂 Projetos

## 🌤️ Pipeline ETL de Dados Climáticos

### Descrição

Pipeline ETL desenvolvido para coleta automática de dados meteorológicos através da API OpenWeatherMap.

O projeto realiza:

* Extração de dados via API REST
* Transformação utilizando Pandas
* Armazenamento em PostgreSQL
* Orquestração através do Apache Airflow

### Tecnologias Utilizadas

* Python 3.11
* Apache Airflow
* PostgreSQL
* Docker
* Pandas
* SQLAlchemy

### Aprendizados

Durante o desenvolvimento deste projeto pratiquei:

* Criação de DAGs no Airflow
* Integração entre containers Docker
* Consumo de APIs externas
* Manipulação de dados com Pandas
* Persistência de dados em PostgreSQL
* Estruturação de pipelines ETL

### Referência

Projeto originalmente baseado no trabalho disponibilizado por Luiza Viana:

https://github.com/vbluuiza/pipeline_etl_weather_data_tutorial_youtube

A reprodução foi realizada exclusivamente para fins educacionais e aprendizado das tecnologias envolvidas.
